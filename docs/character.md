# Character Reference

Source: personal reference photo provided by the site owner (headshot,
blurred glass/concrete building background). Not stored as an image file
here — this is a written brief so the character stays consistent across
every scene (hero, Experience map/jump sequence, any future illustration)
regardless of which tool or illustrator produces the actual art.

## Consistent traits (keep identical in every illustrated scene)

- **Face:** warm brown/tan skin tone, oval face shape, light stubble along
  jaw and upper lip (not a full beard).
- **Hair:** short-to-medium voluminous curly/coily black hair, tighter
  fade on the sides, volume on top.
- **Glasses:** thick black rectangular frames — a defining, always-visible
  feature.
- **Build:** slim/average build, average height.
- **Expression baseline:** warm, confident, slight smile — friendly rather
  than stiff/corporate.

## Variable per scene (adjust to fit context)

- **Hero (idea #3 — working at PC):** smart-casual outfit works better here
  than full formalwear — e.g. rolled-sleeve button-up or plain crewneck,
  seated/working posture, screen glow on face optional for atmosphere.
- **Experience tab (idea #2 — train/plane jump onto Toronto map):** more
  dynamic pose (mid-jump/landing), casual travel-appropriate outfit —
  jacket, maybe a bag — since the original suit read is too formal for an
  action pose.
- Reference photo itself shows a black suit jacket + open-collar white
  shirt, no tie — usable as-is if a more formal variant is ever needed
  (e.g. a static "About" portrait), but not recommended for the animated
  scenes above.

## How to use this brief

- **AI image tools (Midjourney, DALL·E, Leonardo, etc.):** use the original
  photo directly as an image reference / image-to-image input where the
  tool supports it, and pair it with the trait list above as text prompt
  detail (glasses shape, hair volume, stubble, skin tone) to keep results
  consistent across separate generations.
- **Human illustrator:** hand them the reference photo plus this file —
  the "consistent traits" section is what needs to stay locked across every
  commissioned piece.
- **Style choice still open:** flat vector, Lottie-style vector (matches
  the animation approach in [tech.md](tech.md)), or hand-illustrated —
  not yet decided.
